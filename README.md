# 📊 Finance Dashboard

A modern **Financial Dashboard** built using **React + TailwindCSS v4**
with role-based UI, analytics, and responsive design.

------------------------------------------------------------------------

#  Features

## 📈 Dashboard Overview

-   Summary Cards (Total Balance, Income, Expenses)
-   Balance Trend Visualization
-   Spending Breakdown Chart
-   Real-time calculations

## 💳 Transactions Section

-   View transactions list
-   Add new transactions (Admin only)
-   Search transactions
-   Filter by Income / Expense
-   Responsive transaction table
-   Empty state handling

## 👤 Role Based UI

Simulated frontend role switching:

### Viewer

-   View dashboard
-   View transactions
-   View insights

### Admin

-   Add transactions
-   delete transactions
-   Full dashboard access

------------------------------------------------------------------------

## 📊 Insights Section

Financial insights including: - Total Income - Total Expenses - Highest
Spending Category - Monthly Balance Trend - Category Breakdown

------------------------------------------------------------------------

-   Full UI dark mode support
